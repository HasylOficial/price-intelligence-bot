## 🇵🇱 Opis Projektu
Automatyczny system monitorowania cen konkurencji (Business Intelligence). Narzędzie pozwala na automatyczne pobieranie danych o produktach, analizę trendów cenowych oraz generowanie raportów biznesowych w formacie Excel. Rozwiązanie eliminuje godziny ręcznej pracy, pozwalając firmom e-commerce na błyskawiczną reakcję na ruchy rynku.

## 🇺🇸 Project Overview
A robust Price Intelligence & Web Scraping engine designed for e-commerce automation. The bot extracts product data, normalizes pricing information, and generates structured business reports in XLSX format. It features anti-detection mechanisms and a modular architecture for easy scaling.

## ✨ Key Features / Główne Funkcje
- **Multi-Source Scraping:** Modular architecture to support various e-commerce platforms.
- **Anti-Detection Logic:** Custom headers and User-Agent rotation to prevent IP blocking.
- **Data Normalization:** Automatic cleaning of currency symbols and price formatting.
- **Automated Reporting:** Generates professionally formatted Excel files using Pandas.
- **Error Handling:** Robust logging system for tracking script performance and site changes.

## 🛠 Tech Stack
- **Language:** Python 3.9+
- **Libraries:** `requests`, `BeautifulSoup4`, `pandas`, `openpyxl`
- **Data Format:** Export to XLSX / CSV

## 🚀 Quick Start / Instrukcja
1. **Clone the repo:** `git clone https://github.com/TWOJA-NAZWA/price-intelligence-bot.git`
2. **Install dependencies:** `pip install requests beautifulsoup4 pandas openpyxl`
3. **Run the bot:** `python main.py`
